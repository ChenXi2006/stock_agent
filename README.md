# 本项目利用创建的ai-agent对A股个股进行分析
## 目前功能：
* 获取A股当天数据
* 利用ai进行分析
## 技术栈：
* python
* streamlit cloud
* OpenAI
* akshare
## 运行方法：
1.使用anaconda创建python虚拟环境
2.复刻代码（ai的api需要自己写入环境）
3.终端里面运行streamlit run 文件名.py（需要切换到相应的虚拟环境）
